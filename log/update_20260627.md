# 项目更新日志 — 2026-06-27

## 一、Bug修复

### 🔴 Critical: KeyError "Column(s) ['distance_km', 'duration_min'] do not exist"

**根因链路:**
1. `TemporalAnalyzer.__init__()` 接收 DataFrame 后未调用 `_preprocess()`，
   `duration_min` / `distance_km` 列从未被创建
2. `DataCleaner.filter_abnormal_trips()` 在规则5中计算了 `distance_km`/`duration_min`，
   但做完过滤后立即丢弃，只保留了基础OD字段

**修复内容:**
| 文件 | 改动 |
|------|------|
| `src/temporal_analysis.py` | `__init__` 自动调用 `_preprocess()` + 增加 `_preprocessed` 幂等性标记 |
| `src/data_cleaner.py` | `filter_abnormal_trips()` 输出列改为 10 列，保留 `duration_min`, `distance_km`, `avg_speed` |

### 🟡 Major: matplotlib 颜色格式不兼容

- `rgba(255,152,0,0.08)` CSS 语法不被 matplotlib 3.7.5 支持
- 修复：改用 `(R, G, B, A)` 元组格式

### 🟡 Minor: matplotlib CJK 字体警告

- 图表中文标签触发 "Glyph XXXX missing from current font" 警告
- 修复：拷贝 SimHei.ttf 到 matplotlib fonts 目录并重建缓存
